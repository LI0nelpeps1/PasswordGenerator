# PasswordGeneratorKP

Десктопный генератор паролей с графическим интерфейсом на JavaFX. Генерирует криптостойкие пароли, измеряет время генерации и сохраняет пары логин-пароль локально в SQLite.

## Возможности

- **Безопасная генерация** — использует `java.security.SecureRandom` для криптостойкой случайности
- **Настраиваемые наборы символов** — строчные, прописные, цифры, спецсимволы (`!@#$%^&*()-_=+<>?`); поддерживаются латиница и кириллица
- **Настраиваемая длина** — от 1 до 10 000 000 символов
- **Измерение времени генерации** — каждый пароль засекается с точностью до наносекунд
- **Перемешивание Фишера — Йетса** — предотвращает предсказуемый порядок символов
- **Локальное хранение в SQLite** — сохранение и получение паролей по логину
- **JavaFX GUI** — окно с градиентным фоном, флажками, текстовыми полями и кнопками
- **Логирование** — Log4j 2 настроен на вывод в консоль и файл (`logs/app.log`)

## Сборка

```bash
# Сборка проекта
gradlew build

# Создание fat JAR со всеми зависимостями
gradlew shadowJar
```

Итоговые JAR-файлы находятся в `build/libs/`.

## Запуск

```bash
# Через Gradle
gradlew run

# Или напрямую через fat JAR
java -jar build/libs/PasswordGeneratorKP-1.0-SNAPSHOT-all.jar
```

## Тестирование

```bash
gradlew test
```

## База данных

Пароли хранятся в файле `your_database.db` (SQLite) в таблице `users` с колонками `login` (TEXT PRIMARY KEY) и `password` (TEXT NOT NULL).
